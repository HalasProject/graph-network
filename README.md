# Install

## Back-End
```
composer install
```

## Front-End
Install via npm:
```
npm install 
```
Install via yarn:
```
yarn install 
```
Run Vue Server
```
npm run serve
```
# Usage
Create database table & Generate random graphs
```
php artisan migrate && php artisan db:seed
```
Run server
```
php artisan serve
```
Purge all graphs
```
php artisan graphs:clear
```
